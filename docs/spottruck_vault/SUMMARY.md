# Spottruck — Project Build Summary

## Project Overview

**Spottruck** is a logistics auction platform connecting companies needing cargo transport with truck drivers in Argentina. The platform solves the problem of trucks going loaded to Buenos Aires without finding return trips (losing money), while companies/agroquímicas during harvest can't find logistics (losing time).
